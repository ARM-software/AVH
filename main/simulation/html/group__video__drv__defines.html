Video Driver API Definitions.

Video driver provides various definitions to simplify use of its Video Driver API Functions :

  • Channel parameter definitions can be used in VideoDrv_SetFile, VideoDrv_Configure, VideoDrv_SetBuf, VideoDrv_FlushBuf, VideoDrv_StreamStart, VideoDrv_StreamStop, VideoDrv_GetFrameBuf, VideoDrv_ReleaseFrame functions.
  • Event definitions list the events that can be reported via callback function VideoDrv_Event_t.
  • Error code definitions map the status results that can be returned by Video Driver API Functions.
  • Color format definitions list supported color formats used for channel configuration.

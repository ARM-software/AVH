Description

Video Driver API Definitions.

Video driver provides various definitions to simplify use of its Video Driver API Functions :

  • Channel parameter definitions (VIDEO_DRV_IN0..1 and VIDEO_DRV_OUT0..1) can be used in VideoDrv_SetFile, VideoDrv_Configure, VideoDrv_SetBuf, VideoDrv_FlushBuf, VideoDrv_StreamStart, VideoDrv_StreamStop, VideoDrv_GetFrameBuf, VideoDrv_ReleaseFrame functions.
  • Event definitions VIDEO_DRV_EVENT_... list the events that can be reported via callback function VideoDrv_Event_t.
  • Error code definitions (VIDEO_DRV_OK, VIDEO_DRV_ERROR...) map the status results that can be returned by Video Driver API Functions.
  • Color format definitions VIDEO_DRV_COLOR_... list supported color formats used for channel configuration.

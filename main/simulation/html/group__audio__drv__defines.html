Description

Audio Driver API Definitions.

Audio driver provides various definitions to simplify use of its Audion Driver API Functions :

  • Control parameter definitions (AUDIO_DRV_CONTROL_...) can be used in AudioDrv_Control function.
  • Interface definitions (AUDIO_DRV_INTERFACE_...) can be used in AudioDrv_Configure, AudioDrv_SetBuf functions.
  • Error codes definitions (AUDIO_DRV_OK,AUDIO_DRV_ERROR_...) map the status results that can be returned by Audion Driver API Functions.
  • Event definitions AUDIO_DRV_EVENT_... list the events that can be reported via callback function AudioDrv_Event_t.

Triggered to indicate that all the data has been sent (or queued in transmit buffers). The driver is ready for the next transmission.
